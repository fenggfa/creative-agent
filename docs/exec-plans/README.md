# 执行日志

> Agent 的短期记忆区，记录当前任务的进度、状态与决策过程。

## 使用说明

1. 每个 Session 开始时创建新的执行日志
2. 记录关键决策和中间状态
3. 任务完成后归档

## 日志格式

```markdown
# [任务ID] 任务名称

**开始时间**: YYYY-MM-DD HH:MM
**Agent ID**: xxx

## 目标
- [ ] 子任务1
- [ ] 子任务2

## 执行记录
| 时间 | 动作 | 结果 |
|------|------|------|
| ... | ... | ... |

## 决策记录
- 为什么选择方案 A？
- ...

## 遗留问题
- [ ] 待解决1
```
